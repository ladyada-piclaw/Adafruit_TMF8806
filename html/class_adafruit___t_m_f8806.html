void Adafruit_TMF8806::setGPIOMode ( uint8_t  gpio,
tmf8806_gpio_mode_t  mode 
)

Set GPIO mode (applied on next startMeasuring() call)

GPIO configuration is part of the measurement command and takes effect when startMeasuring() is called. There is no standalone GPIO command.

Note
GPIO0 is used for I/O voltage auto-detection at startup (EN rising edge). Ensure any external pull-up on GPIO0 does not conflict with the desired output mode.
Parameters
gpioGPIO number (0 or 1)
modeGPIO mode
